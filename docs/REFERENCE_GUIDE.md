# Hướng Dẫn Prompt Tham Khảo
## File này chỉ dành cho người dùng, không phải cho AI đọc trực tiếp

### Mục đích
File này giúp bạn soạn yêu cầu rõ ràng khi làm việc với AI Agent. Bạn dùng nó để tham khảo cách đặt câu hỏi, không cần copy toàn bộ vào AI.

---

## 1. Nếu bạn muốn lập kế hoạch dự án

```
[THAM KHẢO] Tôi đang nghiên cứu dự án VBA/Python.

Mục tiêu:
- [Mô tả kết quả cần đạt]
- [Dữ liệu vào và dữ liệu ra]
- [Yêu cầu chất lượng hoặc hiệu năng nếu có]

Hãy đề xuất cho tôi:
1. Cấu trúc module phù hợp
2. Quy trình làm việc đơn giản
3. Những rủi ro hoặc phần nên tránh
```

---

## 2. Nếu bạn muốn tạo module mẫu

```
[THAM KHẢO] Tôi cần module cho [VBA/Python].

Mục đích module: [Mô tả chức năng]

Các hàm cần thiết:
1. [TênHàm](tham số) - [mô tả]
2. [TênHàm](tham số) - [mô tả]

Vui lòng cung cấp:
- Template module có comment
- Ví dụ cách sử dụng
- Xử lý lỗi hoặc kiểm tra đầu vào
```

---

## 3. Nếu bạn cần sửa lỗi logic

```
[SỬA LỖI] Tôi có vấn đề logic trong code [VBA/Python].

Vấn đề:
- [Mô tả hành vi sai]
- [Kết quả mong muốn và kết quả hiện tại]

Code liên quan:
[Chỉ dán phần code liên quan, khoảng 20 dòng]

Vui lòng cho tôi biết:
1. Tại sao sai
2. Cách sửa
3. Điều cần tránh lần sau
```

---

## 4. Nếu bạn cần cập nhật tài liệu tham khảo

```
[TÀI LIỆU] Tôi muốn sửa file tham khảo.

File: [ví dụ docs/README.md hoặc docs/QUICK_START_GUIDE.md]
Thay đổi: [Mô tả cụ thể]
Lý do: [Vì sao thay đổi lại tốt hơn]
```

---

## Lưu ý chung

- Không cần đưa toàn bộ nội dung file vào AI.
- Càng ngắn gọn và cụ thể càng tốt.
- Chỉ gửi những thông tin cần thiết.
- Dùng mẫu này để suy nghĩ trước khi gõ yêu cầu.

---

## Kết luận
File này là tài liệu cho bạn đọc. Hãy dùng nó để chuẩn bị prompt, rồi gõ yêu cầu vào AI sao cho ngắn, rõ ràng và có đủ context.
